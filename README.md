# Lifestyle-Related-Health-Risk-Prediction
Machine Learning project focused on predicting lifestyle-related health risks using demographic and behavioral data such as age, BMI, physical activity, and other factors. The project involved data preprocessing, feature engineering, model training, and evaluation to identify key health risk indicators and support preventive healthcare insights.

## Project Overview
This project aims to explore how lifestyle factors can influence health risk prediction using supervised machine learning techniques. The workflow included data preprocessing, feature engineering, model training, and performance evaluation.

## Objectives
- Analyze lifestyle-related health data
- Identify key health risk indicators
- Build predictive machine learning models
- Explore AI applications in preventive healthcare

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Machine Learning Workflow
1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Model Evaluation and Accuracy Testing

## Results
The model achieved strong predictive performance in identifying potential health risks based on lifestyle-related factors.

## Future Improvements
- Improve model generalization using larger real-world datasets
- Deploy the model using Streamlit or Flask
- Integrate additional healthcare indicators for more accurate predictions
